# Quick Start

This section walks you through the essential steps for getting started with the generative AI hub in SAP AI Core. You'll authenticate, retrieve your orchestration deployment URL, and make your first model call using the Harmonized API.

> ### Tip:  
> If you want a guided, hands-on environment, you can explore the generative AI hub using the 30-day free trial. For more information, see [Try now: 30-Day Basic Trial](https://www.sap.com/products/artificial-intelligence/generative-ai-hub-trial.html).



## Prerequisites

-   You've completed the tasks described in [Initial Setup](https://help.sap.com/viewer/2d6c5984063c40a59eda62f4a9135bee/CLOUD/en-US/38c4599432d74c1d94e70f7c955a717d.html "Get started with SAP AI Core using the standard procedures for the SAP BTP, Cloud Foundry environment or Kyma environment.") :arrow_upper_right:.
-   You're using the extended service plan, which is required for the generative AI hub.

## Procedure

To get started, complete the following steps:

1.  [Get an Auth Token](get-an-auth-token-0808d42.md): Obtain an authentication token to securely access SAP AI Core services. You use this token in all subsequent API calls.
2.  [Get An Orchestration Deployment URL](get-an-orchestration-deployment-url-ec7c703.md): Retrieve the URL of your Orchestration service deployment. This URL is required for making inference requests through the Harmonized API.
3.  [Consume Models with the Harmonized API](consume-models-with-the-harmonized-api-2392d9a.md): Use the Harmonized API to make inference calls without needing to manage individual model deployments. The Harmonized API provides a uniform interface for interacting with supported large language models.
4.  [Next Steps](next-steps-6106eed.md): After completing an initial model call, you can explore grounding, content filtering, data masking, and additional orchestration capabilities.

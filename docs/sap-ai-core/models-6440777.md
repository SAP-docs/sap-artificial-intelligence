# Models

SAP AI Core provides access to a comprehensive collection of pre-trained machine learning models designed to accelerate your AI initiatives. These models serve as ready-to-use building blocks that can be integrated into your applications and business processes, eliminating the need to develop complex AI capabilities from scratch.

The generative AI hub model library spans multiple model types and use cases. For more information, see the following:
